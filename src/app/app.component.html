POC: amplify-theme-provider

This POC shows that we can implement a amplify-theme-provider that:

  • parses given theme object, convert it to an object of CSS variables, and have it cascade downwards.
  • allows nested providers, and have scoped styles by using the nearest parent's style.

    • Demo

    This demo renders three buttons.

    The first one is not in a provider, and uses the default theme. The second one is in a provider, that uses a purple theme. The third one is in a nested provider, that uses a green theme.
